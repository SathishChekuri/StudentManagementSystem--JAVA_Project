## Student Management System (Java)

## Project Description
The Student Management System is a menu-driven console application developed using Core Java.
It allows users to add, view, and search student records efficiently.
The project demonstrates strong understanding of OOP concepts and Java Collections.
## Features

Add new student details


View all students in tabular format

Search student by ID

Menu-driven and user-friendly

Handles multiple student records dynamically

## Technologies Used

Java (Core Java)
